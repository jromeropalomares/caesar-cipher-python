# Caesar Cipher Encoder/Decoder (Python)

## Overview
This project is a Python-based Caesar Cipher tool that allows users to encode and decode text using rotation-based substitution.

## Features
- Encode and decode messages using a rotation value
- Preserves punctuation, spaces, and symbols
- Handles both uppercase and lowercase letters
- Brute-force decoding mode to find the correct rotation

## Example
encode, decode, quit?: encode
Encode input: Hello, World!
Rotation: 3

Output:
Khoor, Zruog!

## How to Run
1. Download the file
2. Run in terminal:
   python caesar_cipher.py
